# Mailhog Docker Configuration

This is my mailhog configuration for local development.

## Run it

`docker-compose up -d`

| Service | URL                                                    |
| ------- | ------------------------------------------------------ |
| HTTP    | [http://localhost:20001](http://localhost:20001)       |
| SMTP    | `localhost` or `docker.for.mac.localhost` port `20002` |
